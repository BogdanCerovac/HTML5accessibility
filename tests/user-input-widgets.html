ARIA User Input Widget role tests - Firefox 30 + Assistive Technology

Results detail what is announced when each element with the role indicated in brackets receives focus.

Testing conducted July 2014.

Software used for testing

  • Windows 8.1
  • Mac OSX ...
  • Firefox 30
  • Safari 7
  • aViewer (for MSAA information)
  • JAWS 15
  • Window Eyes 8.1
  • VoiceOver
  • NVDA latest

Notes for Table 1

  • JAWS 9 and 10 change to PC cursor mode (Insert+Z).
  • Jaws 10: Turn off 'auto forms mode' by unchecking it in Configuration Manager > Set Options > Forms Mode Options.
  • For Window eyes change out of browse mode (ctr+shift+A).
  • NVDA enable Virtual buffer pass through (NVDA key + spacebar)
  • For all Assistive software use the default verbosity settings
  • Tab to each focusable element in the table
  • For "radio group", tab to it, then use the arrow key to change the selection.
